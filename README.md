# SBU-Hacks-Hackathon-Project

Name of Project: Music-Choices

Music Choices is able to provide a random song based on the user's preferences. There are three main topics you can choose from, holiday, genre, or mood. Then it goes into greater details to customize to what the user wants. Then it suggest a song for the user to check out.
